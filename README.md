# 项目结构

由于h5的音频播放无法在后台持续播放，所以该项目的结构是：所有的页面都是通过vue完成，播放功能是使用原始的播放器实现。所以含有大量的原生交互功能。

所有的交互逻辑都在InteractionTools这个类中

# 该项目的特点
